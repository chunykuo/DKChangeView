# DKChangeView

A demo that showcases multiple ways to transition between view controllers.

This project uses a Storyboard configured with two UIViewControllers, presented in two variants (with and without a Navigation Controller) to highlight the differences.

Navigate to a new ViewController in three ways:

1. performSegue
2. present (modal presentation)
3. push (via a navigation controller)

## 中文說明

DKChangeView

以 Storyboard 配置兩個 UIViewController，並且使用兩種版本呈現(有無 Navigation)。
來示範多種切換畫面（ViewController）的方法，展示出差異

以3種方式前往新的 ViewController：

1. performSegue
2. present（模態呈現）
3. push（使用導航控制器）
